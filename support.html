STEP File Viewer

Local • Private • Professional

Purpose

STEP File Viewer is a lightweight 3D model tool, no registration, no uploads, no internet required. Everything works offline, locally, and privately on your device.

Key Features

  • Import: STL, OBJ, 3MF, GLB, USDZ, TDS, AMF, PLY, PDC, XYZ, VOX, VTK, VTP, and FBX
  • Export: STL, OBJ, GLB, USDZ, and GLTF
  • Capture: Model screenshots for easy export or sharing
  • Edit: Move, rotate, scale, delete, and undo tools for model objects
  • Pure: No ads. No tracking. No data collection.

Technical Specifications

• Requires iOS 16, macOS 12 or later.
• Optimized for high performance and resource efficiency.
• Complete data privacy with zero network requirements.
